

# 🚀 CraftAI Project
A .NET-based application that integrates AI functionality with a structured backend system. This project appears to include controller logic, view handling, and Docker support for deployment.
---
## 📂 Project Structure

├── CraftAi.cs            # Core AI logic / main module
├── Homecontroller.cs     # Controller handling main routes
├── View_start.cs         # View initialization
├── Viewimport.cs         # View imports / dependencies
├── Launch settings       # App launch configuration
├── Docker               # Docker configuration file

---
## ⚙️ Features
- 🧠 AI-based processing (via `CraftAi.cs`)
- 🌐 MVC-style architecture
- 🐳 Docker support for containerized deployment
- ⚡ Lightweight and modular code structure

## 🛠️ Technologies Used
